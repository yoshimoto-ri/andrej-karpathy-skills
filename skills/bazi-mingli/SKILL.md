# 八字命理分析技能

> 命盤呈現的是**傾向、結構、課題與機會**，不是絕對命定。每個結論必須有推演過程支撐。

---

## 觸發條件

輸入包含以下任一詞彙時啟動：八字、四柱、命盤、排盤、算命、天干、地支、日主、十神、大運、流年、用神、格局、五行、命理

---

## 分析流程（依序執行）

1. **收集命造資訊** → 排出四柱
2. **判斷日主強弱** → 確認格局
3. **推導用神喜忌**
4. **配置十神**
5. **排大運** → 分析流年
6. **綜合解讀**（性格、事業、婚姻、財運、健康）

### ◆ 月干支（五虎遁年法）

- 換月節點為**節氣**，**非農曆初一**（最常見錯誤）
- 月支固定：寅(1月)卯(2月)辰(3月)巳(4月)午(5月)未(6月)申(7月)酉(8月)戌(9月)亥(10月)子(11月)丑(12月)
- 月干由年干推算：

| 年干 | 寅月干 | 規律（每月順推） |
|------|--------|----------------|
| 甲、己 | 丙 | 丙寅→丁卯→戊辰→己巳→庚午→辛未→壬申→癸酉→甲戌→乙亥→丙子→丁丑 |
| 乙、庚 | 戊 | 戊寅→己卯→庚辰→辛巳→壬午→癸未→甲申→乙酉→丙戌→丁亥→戊子→己丑 |
| 丙、辛 | 庚 | 庚寅→辛卯→壬辰→癸巳→甲午→乙未→丙申→丁酉→戊戌→己亥→庚子→辛丑 |
| 丁、壬 | 壬 | 壬寅→癸卯→甲辰→乙巳→丙午→丁未→戊申→己酉→庚戌→辛亥→壬子→癸丑 |
| 戊、癸 | 甲 | 甲寅→乙卯→丙辰→丁巳→戊午→己未→庚申→辛酉→壬戌→癸亥→甲子→乙丑 |

### ◆ 日干支

使用萬年曆查閱，或使用以下基準推算：
- 基準：2000年1月1日 = 己卯日（干支序號 16）
- 每日加 1，滿 60 歸零循環
- 逢閏年 2 月後需加 1 日

## 第三步：判斷日主強弱

**判斷順序：月令（50%）→ 通根 → 透干 → 綜合**

1. 查月令五行旺衰表 → 日主在當月屬旺/相/休/囚/死？
2. 數各地支藏干中同五行的根（本氣×100%，中氣×60%，餘氣×30%）
3. 天干中印星、比劫是否透出？
4. 整體生扶 vs 剋洩力量比較

| 結論 | 條件 |
|------|------|
| 身強 | 月令得旺相 + 生扶多 |
| 身弱 | 月令休囚死 + 生扶不足 |
| 身中和 | 兩力接近均衡 |
| 從格 | 日主極弱且無一根（或極旺無一剋洩） |
