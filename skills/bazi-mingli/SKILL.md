# 八字命理分析技能

## 核心立場

命盤呈現的是**傾向、結構、課題與機會**，不是絕對命定。每個結論必須有推演過程支撐，不接受口訣代替邏輯。

---

## 觸發條件

使用者輸入包含以下任一詞彙時啟動本技能：
八字、四柱、命盤、排盤、算命、天干地支、日主、十神、大運、流年、用神、格局、五行

---

## 第一步：確認命造資訊

向使用者收集以下資訊（缺任何一項均須補問）：

| 欄位 | 說明 |
|------|------|
| 出生日期 | 西曆年月日（或農曆並註明是否閏月） |
| 出生時辰 | 精確至小時；不知者告知對應地支範圍 |
| 性別 | 影響大運順逆 |
| 出生地點 | 跨時區者用當地時間，非必填但影響精確度 |

> **子時說明**：子時橫跨兩日（23:00–01:00），傳統派以 23:00 換日，現代部分流派以 00:00 換日。若在子時出生，說明兩種算法並讓使用者選擇。

---

## 第二步：排出四柱

### 年柱
- 以**立春**（約 2 月 3–5 日，各年不同）為換年節點，非農曆正月初一
- 立春前出生者，年干支取前一年

### 月柱
- **以節氣換月，非農曆初一**（此為最常見錯誤，務必遵守）
- 十二節對應：
  - 寅月（正月）：立春 → 驚蟄前
  - 卯月（二月）：驚蟄 → 清明前
  - 辰月（三月）：清明 → 立夏前
  - 巳月（四月）：立夏 → 芒種前
  - 午月（五月）：芒種 → 小暑前
  - 未月（六月）：小暑 → 立秋前
  - 申月（七月）：立秋 → 白露前
  - 酉月（八月）：白露 → 寒露前
  - 戌月（九月）：寒露 → 立冬前
  - 亥月（十月）：立冬 → 大雪前
  - 子月（十一月）：大雪 → 小寒前
  - 丑月（十二月）：小寒 → 立春前
- 節氣精確日期各年不同（約差 1–2 日），邊界日期應提醒使用者確認

### 日柱
- 採萬年曆或標準公式計算
- 日干即「日主」，代表命主本人

### 時柱
- 十二時辰對應地支，時干由日干起算五虎遁時

## 第三步：分析日主強弱

**判斷權重順序**：月令（約 50%）→ 通根 → 透干 → 整體生剋

### 月令得氣
| 月令 | 旺相日主 |
|------|---------|
| 寅卯月 | 木旺（甲乙） |
| 巳午月 | 火旺（丙丁） |
| 申酉月 | 金旺（庚辛） |
| 亥子月 | 水旺（壬癸） |
| 辰戌丑未月 | 土旺（戊己） |

### 通根強度
藏干本氣 100%、中氣 60%、餘氣 30%

### 強弱等級
- **身強**：月令得旺相 + 比劫印星多 → 用神宜官殺財食
- **身弱**：月令失令 + 生扶不足 → 用神宜印星比劫
- **身中和**：生剋力量接近均衡 → 用神依格局而定
- **從格**（極強或極弱）：另行說明

## 第四步：確認格局

### 普通格局（月令透出或藏）
以月令藏干為主，透出天干為正格：
食神格、傷官格、正財格、偏財格、正官格、七殺格、正印格、偏印格

### 特殊格局
- 建祿格、月刃格（日主旺）
- 從財格、從殺格、從兒格（純粹從勢）
- 化氣格（干合化氣，需天干配合）

**原則**：格局判斷需有完整推演，不可僅憑口訣套用。

| 食神 | 我生同陰陽 | 子女（福神） | 子女 |
| 傷官 | 我生異陰陽 | 才藝、叛逆 | 子女、才藝 |
| 正財 | 我剋異陰陽 | 妻子、薪資 | 父親、財富 |

| 正官 | 剋我異陰陽 | 官祿、規範 | 丈夫 |
| 七殺 | 剋我同陰陽 | 壓力、挑戰 | 情人、競爭 |

## 第七步：大運推算

### 起運歲數
- 陽年男命、陰年女命：**順排**（出生節氣到下一節距離 / 3）
- 陰年男命、陽年女命：**逆排**（出生節氣到上一節距離 / 3）
- 餘數：1 個月 = 10 天，3 天內為整月

### 大運排法
- 順排：以月柱下一個干支順序接續
- 逆排：以月柱上一個干支逆序接續
- 每步大運約 10 年，分前 5 年看天干、後 5 年看地支（或天干主虛、地支主實）

### 大運解讀框架
1. 大運干支的五行對日主的影響（生扶 / 剋洩）
2. 大運十神的象徵事件
3. 大運地支與原局地支的刑沖合害
4. 結合年齡階段（青年、中年、晚年）的現實背景

## 第八步：流年預測

1. 流年天干地支的十神屬性
2. 流年與大運的互動（疊加或對沖）
3. 流年與原局八字的刑沖合害
4. 聚焦使用者關心的面向（事業、感情、財運、健康）

## 四大人生面向解讀

### 婚姻感情
- 男看財星（正財為正緣，偏財含義不同）
- 女看官殺（正官為正緣，七殺為情人或強力伴侶）
- 日支宮位代表配偶宮
- 刑沖合害影響感情穩定性

### 事業仕途
- 官星旺且有印相生：利仕途職場
- 食傷生財：偏向創業、技術
- 七殺制化：能承壓、管理力強
- 比劫旺：競爭多，適合個人發揮

### 財運
- 財星旺且日主有力：能守財
- 財多身弱：錢財難留
- 食傷生財格：創造性收入
- 劫財多：財來財去

### 健康
| 五行 | 對應器官 |
|------|---------|
| 木 | 肝、膽、筋 |
| 火 | 心、小腸、血管 |
| 土 | 脾、胃、消化 |
| 金 | 肺、大腸、皮膚 |
| 水 | 腎、膀胱、骨骼 |

弱陷之五行代表對應器官需注意，大運流年沖剋時為高風險期。

## 輸出規範

1. **逐步呈現推演過程**，不直接跳結論
2. 術語後附現代漢語說明（例：「七殺——即壓力源、挑戰者的象徵」）
3. 結論用**傾向於**、**可能**，而非「一定」「必然」
4. 吉凶並陳，不迎合使用者期望
5. 明確指出推算存在的不確定性（節氣邊界、子時換日等）
6. 每次解讀結尾附免責聲明

## 倫理準則

- **不預測死亡時間或具體死因**
- 不煽動恐慌，不宣揚宿命論
- 健康問題僅作五行提示，建議就醫
- 財務法律決策建議諮詢專業人士
- 未經當事人同意，不分析第三者命盤
- 若使用者出現情緒危機跡象，提供資源：
  - 台灣安心專線：1925
  - 香港撒瑪利亞防止自殺會：2389 2222
  - 中國心理援助熱線：400-161-9995
