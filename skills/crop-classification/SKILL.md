# 農產品分類查詢技能

依據農委會作物分類資料，快速回答作物的分類歸屬。

## 觸發條件

使用者輸入包含以下任一情境時啟動本技能：

- 「○○屬於哪一類 / 哪一科 / 什麼分類」
- 「○○是蔬菜還是水果 / 是哪個大分類」
- 「列出 ○○類 / ○○科 底下有哪些作物」
- 在 crop-records 系統中新增或查詢作物時，需要對應的分類資訊

## 分類階層

資料採五層階層，由大到小：

| 層級 | 欄位 | 說明 | 範例 |
|------|------|------|------|
| 大分類 | PLV1_NAME | 最上層類別 | 蔬菜類 |
| 次分類 | PLV2_NAME | 用途/型態次群 | 果菜類 |
| 科 | PLV5_NAME | 植物科別 | 茄科 |
| 作物 | PLV3_NAME | 作物本身 | 甜椒 |
| 品種 | PLV4_NAME | 品種/型態 | 水果彩椒 |

> 注意欄位編號與階層順序不一致：呈現時請依「PLV1 → PLV2 → PLV5 → PLV3 → PLV4」排列。

八大大分類及其資料量：花卉類、蔬菜類、果樹類、農產品加工類、特用作物類、雜糧類、稻米類、其他作物類。

## 查詢方式

使用 `scripts/lookup.py` 查詢，資料位於 `data/crop-classification.json`。

```bash
# 查單一作物的分類歸屬（支援作物名、全名、別名關鍵字）
python3 scripts/lookup.py 甜椒
python3 scripts/lookup.py 番茄

# 列出某科底下的作物
python3 scripts/lookup.py --family 茄科

# 列出某大分類底下的次分類
python3 scripts/lookup.py --cat 蔬菜類
```

回應使用者時，請以分類路徑呈現，例如：

> 甜椒 → 蔬菜類（大分類）> 果菜類（次分類）> 茄科（科）

## 注意事項

- **一作物可能有多重歸屬**：少數作物因用途不同分屬不同大分類（例：高粱可為「雜糧類>穀類」也可為「花卉類」；香茅可為「特用作物類>香料類」也可為「花卉類」）。查到多筆時請全部呈現，勿任意挑一個。
- **加工品另立分類**：同名的加工製品歸在「農產品加工類」（例：番茄製品、高粱製品），與生鮮作物分開。
- 別名（ALIAS_CNAME）也納入比對，例如「飼料玉米」可對應到玉米相關項目。
- 查無結果時如實告知，不要臆測分類。
